Represents the configuration for the MinIOStorageTrailer. This implements usage with kotlinx.serialization so you don't have to create boilerplate code for this.

Parameters

endpoint

The endpoint to connect to MinIO. You can use:

    - The `minio.endpoint` JVM argument (`-Dminio.endpoint=...`)
    - The `MINIO_ENDPOINT` environment variable (`MINIO_ENDPOINT=... java -jar ./something.jar`)
    - Set this property to a URI.
