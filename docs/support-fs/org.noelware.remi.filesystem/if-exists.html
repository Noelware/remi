fun <T> File.ifExists(body: File.() -> T): T?
Content copied to clipboard

Extension for File if the File exists, then call the body function, or return null if it doesn't exist. It is present in utilities like FilesystemStorageTrailer.open and such.

Return

The result of the body block, or null.

Parameters

body

The body to run if this File exists.
