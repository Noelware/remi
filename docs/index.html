Library to handling files for persistent storage with Google Cloud Storage and Amazon S3-compatible server, made in Kotlin!

Why is this built?

This was built in the mind of Arisu and charted to store any piece of data into different providers without repeating code and so on, and so forth. This is a simple library to take care of that!

We support any Amazon S3 compatible servers, Google Cloud Storage, MinIO, and the local filesystem!

The library is split into five modules:

  • ./bom - Bill of Materials for Remi

  • ./core - Core abstractions that is used for the storage units.

  • ./support-fs - Support for using the local filesystem

  • ./support-gcs - Support for using Google Cloud Storage

  • ./support-s3 - Support for using any compatible Amazon S3 service like Wasabi and MinIO!
